NC News API

A RESTful API for accessing news articles, comments, topics and users.

  • Topics
  • Articles
  • Comments
  • Users

Topics

GET /api/topics

Returns a list of all topics.

Example Response

{
  "topics": [
    { "slug": "coding", "description": "Code is love, code is life" }
  ]
}

Articles

GET /api/articles

Returns a list of all articles, sorted by date descending by default.

Queries

Query Accepted Values Default
sort_by created_at, author, title, article_id, topic, votes, comment_count created_at
order asc, desc desc
topic any valid topic slug none

Example Response

{
  "articles": [
    {
      "article_id": 1,
      "title": "Running a Node App",
      "topic": "coding",
      "author": "jessjelly",
      "created_at": "2020-11-07T06:03:00.000Z",
      "votes": 0,
      "article_img_url": "https://example.com/image.jpg",
      "comment_count": 8
    }
  ]
}

GET /api/articles/:article_id

Returns a single article by its ID.

Example Response

{
  "article": {
    "article_id": 1,
    "title": "Running a Node App",
    "topic": "coding",
    "author": "jessjelly",
    "body": "Well, the steps are simple...",
    "created_at": "2020-11-07T06:03:00.000Z",
    "votes": 0,
    "article_img_url": "https://example.com/image.jpg",
    "comment_count": 8
  }
}

PATCH /api/articles/:article_id

Updates the vote count of an article. Pass a positive or negative integer to increment or decrement.

Request Body

{ "inc_votes": 1 }

Example Response

{
  "article": {
    "article_id": 1,
    "votes": 1,
    ...
  }
}

Comments

GET /api/articles/:article_id/comments

Returns all comments for a given article, most recent first.

Example Response

{
  "comments": [
    {
      "comment_id": 10,
      "body": "Great article!",
      "article_id": 1,
      "author": "jessjelly",
      "votes": 4,
      "created_at": "2020-11-07T06:03:00.000Z"
    }
  ]
}

POST /api/articles/:article_id/comments

Adds a new comment to an article.

Request Body

{
  "username": "jessjelly",
  "body": "What a great read!"
}

Example Response

{
  "comment": {
    "comment_id": 19,
    "body": "What a great read!",
    "article_id": 1,
    "author": "jessjelly",
    "votes": 0,
    "created_at": "2024-01-01T00:00:00.000Z"
  }
}

DELETE /api/comments/:comment_id

Deletes a comment by its ID. Returns no content on success.

Response

Status: 204 No Content

Users

GET /api/users

Returns a list of all users.

Example Response

{
  "users": [
    {
      "username": "jessjelly",
      "name": "Jess Jelly",
      "avatar_url": "https://example.com/avatar.jpg"
    }
  ]
}
